# Task Manager API

RESTful API built with ASP.NET Core that allows managing tasks using CRUD operations.

## 🚀 Features
- Create tasks
- Retrieve tasks
- Update tasks
- Delete tasks

## 🛠 Technologies
- C#
- ASP.NET Core Web API
- .NET

## 📌 Endpoints

- GET /api/tasks
- POST /api/tasks
- PUT /api/tasks/{id}
- DELETE /api/tasks/{id}

## 📷 Testing
Tested using Swagger

## ⚙️ Run Locally
1. Clone the repository
2. Open in Visual Studio
3. Run the project
4. Navigate to /swagger
